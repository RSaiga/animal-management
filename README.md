### animal-management

#### warファイルの作成

```sh
$ ./gradlew clean build
```

#### warファイル作成場所

```
build/libs/animal-management-web.war
```

#### mysql起動
```sh
$ cd docker/
$ docker-compose up -d
```

#### mysql停止
```sh
$ docker-compose dawn
```
